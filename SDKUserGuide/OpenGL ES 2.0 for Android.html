Contents

The PowerVR Insider SDK includes tutorials, source code, documentation, platform abstraction frameworks and a highly integrated suite of tools. To get started, visit the Installation page to complete the SDK installation.

To get started with the SDK, see the Examples page. The examples range from initialising the API through to complex 3D graphics techniques. Many of these take advantage of our PVRShell abstraction layer and PVRTools library which are both available as source libraries.

To aid in the development process we also provide various Documentation. See the Documentation page for a full list of user manuals, whitepapers, specifications and quick start guides.

Developers can join the PowerVR Insider programme and interact with the community through online forums at www.powervrinsider.com.
